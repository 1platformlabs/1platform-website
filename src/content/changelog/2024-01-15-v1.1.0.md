## New Features

- **Payment Processing** — Accept online card payments through the API. Create transactions, track payment status, and receive webhook notifications.
- **NFC Tap-to-Pay** — Card-present payments via mobile app for in-person transactions.
- **Electronic Invoicing (FEL)** — Generate compliant electronic invoices automatically from transactions.
- **Business Management** — Create and manage business entities and branches for invoicing.

## Improvements

- Enhanced content generation pipeline with faster processing times
- Improved keyword extraction with cannibalization detection
- Updated API documentation with interactive examples

## API Changes

- New endpoints: `/users/transactions`, `/businesses`, `/businesses/{id}/invoices`
- New webhook: `POST /webhooks/transactions` for payment status callbacks
- Rate limit adjustments for content generation (5 req/min)
