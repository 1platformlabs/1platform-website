## API Developer Documentation

- **Paid Onboarding flow** — New integration flow documenting how to register user accounts via payment. Create a checkout session, poll for completion, and retrieve the API key — all through 2 endpoints (`POST /onboarding/`, `GET /onboarding/status/{session_id}`).
- **Google AdSense earnings** — Updated earnings overview and by-page response schemas to match the latest API. Added `impressions`, `clicks_ctr`, `cost_per_click`, and `change_percent`/`trend` fields to the earnings overview. Added `rpm`, `clicks`, `total_earnings`, and `total_pages` to the earnings by-page response.
